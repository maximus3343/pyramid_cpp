Scale, translate, and rotate a 3D pyramid, in OpenGL.

## Controls

- **W**: Move up
- **S**: Move down
- **A**: Move left
- **D**: Move right
- **Q**: Rotate counter-clockwise around the Z-axis
- **E**: Rotate clockwise around the Z-axis
- **R**: Scale up
- **F**: Scale down
